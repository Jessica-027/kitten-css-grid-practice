Sobre Este Projeto

Perfeito para Aprender CSS Grid Moderno

Esta landing page foi projetada especificamente para desenvolvedores front-end que desejam praticar layouts CSS Grid. Cada seção usa diferentes configurações de colunas, desde divisões simétricas 6/6 até layouts assimétricos 8/4.

  • Sistema de grid responsivo de 12 colunas
  • Estrutura HTML limpa e semântica
  • Efeitos de hover e transições suaves
  • Design responsivo mobile-first
